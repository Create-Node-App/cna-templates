This is a **sample blog post** shipped with the Create Awesome Node App Astro starter.

Content lives in `src/content/blog/` and is validated at build time via `src/content.config.ts`. Edit this file or add new Markdown entries — the `/blog` index and detail routes will pick them up automatically.

## Next steps

- Add frontmatter fields in `content.config.ts` when you need tags, authors, or hero images.
- Create MDX entries if you want components inside posts (`*.mdx` is already supported by the glob loader).
- Remove `draft: true` posts from production builds by filtering in your page queries.

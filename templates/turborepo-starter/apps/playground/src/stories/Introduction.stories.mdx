# Create Awesome Node App — Monorepo Playground

This Storybook app documents shared packages in your Turborepo monorepo. Use it to develop and showcase UI components from `packages/` in isolation from application code.

## What's in this repo

- **`packages/ui`** — shared React components (with Vitest tests)
- **`apps/web`** — Vite + React app that imports workspace packages
- **`apps/playground`** — this Storybook instance

## Next steps

1. Add components to `packages/ui` and export them from `index.tsx`
2. Write stories in `src/stories/` to document components and states
3. Import shared components in `apps/web` to validate real app integration

Scaffold additional apps (APIs, extensions, frameworks) with [create-awesome-node-app](https://www.npmjs.com/package/create-awesome-node-app) and place them under `apps/`.

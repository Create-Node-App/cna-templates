# Member Guide Overview

Next.js SaaS AI Template is an AI-native platform for skills management, career growth, and team development. As a **member**, you can manage your profile, assess your skills, track learning, set OKRs, participate in performance reviews, and connect with colleagues.

## What You Can Do

| Area                       | What's available                                                      |
| -------------------------- | --------------------------------------------------------------------- |
| **Dashboard**              | Stats, quick actions, AI recommendations, activity feed               |
| **Profile & Settings**     | Edit profile, avatar, connect GitHub/LinkedIn, timezone, pronouns     |
| **Skills Self-Assessment** | Assessment wizard, skill levels (1–5), evidence, verification         |
| **Interests & Growth**     | Declare interests, growth path recommendations, suggested roadmaps    |
| **Learning**               | Assignments from manager/instructor, roadmaps, quizzes, progress      |
| **OKRs**                   | Objectives (career, skill, project, learning), key results, check-ins |
| **Performance**            | Self-assessments, upward assessments (evaluate your manager)          |
| **Projects**               | View assignments, roles, dates, clients, project members              |
| **People Finder**          | AI search by skill/capability, match scores, natural language         |
| **Knowledge Base**         | Role profiles, roadmaps, trainings, capability docs                   |
| **Directory & Org Chart**  | Browse team, hierarchy view, search by name/skill                     |
| **Feedback & Recognition** | Give feedback, send recognition/praise with categories                |
| **AI Assistant**           | Chat, search people by skills, recommendations                        |

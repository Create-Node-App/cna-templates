# Dashboard

Tu Dashboard es tu base. Muestra estadísticas clave, acciones rápidas y actividad reciente para que puedas estar al día con lo que importa.

## Tarjetas de estadísticas

En la parte superior del Dashboard vas a ver tarjetas resumen con métricas relevantes para tu tenant. Estas tarjetas dan una visión rápida de tu actividad e ítems pendientes.

> **Tip:** Hacé clic en una tarjeta para ir a la sección relacionada.

## Acciones rápidas

Las acciones rápidas te permiten iniciar tareas habituales sin salir del Dashboard:

- Ver miembros del equipo
- Abrir el **Asistente de IA** para una consulta
- Navegar a configuración de admin (si tenés permisos)

Usalas para acceder rápidamente a los flujos de trabajo más comunes.

## Feed de actividad

El feed de actividad muestra eventos recientes relevantes para vos, como:

- Nuevos miembros que se suman al tenant
- Cambios de configuración
- Notificaciones del sistema

Desplazate para ver más. Usalo para estar al tanto de cambios y seguimientos.

### Cómo aprovechar al máximo tu Dashboard

1. **Revisá las estadísticas** — Mirá las métricas clave para ver en qué punto están las cosas.
2. **Usá las acciones rápidas** — Accedé a tareas comunes sin navegar menús.
3. **Revisá el feed** — Ojeá la actividad reciente por si algo requiere atención.

> **Tip:** Visitá tu Dashboard regularmente para estar informado de la actividad de tu tenant.

# Guía del miembro

Next.js SaaS AI Template es una plataforma nativa de IA para gestión de habilidades, crecimiento profesional y desarrollo de equipos. Como **miembro**, podés gestionar tu perfil, evaluar tus habilidades, seguir tu aprendizaje, definir OKRs, participar en evaluaciones de desempeño y conectar con colegas.

## Qué podés hacer

| Área                              | Qué tenés disponible                                                                |
| --------------------------------- | ----------------------------------------------------------------------------------- |
| **Dashboard**                     | Estadísticas, acciones rápidas, recomendaciones de IA, feed de actividad            |
| **Perfil y configuración**        | Editar perfil, avatar, conectar GitHub/LinkedIn, zona horaria, pronombres           |
| **Autoevaluación de habilidades** | Asistente de evaluación, niveles de habilidad (1–5), evidencia, verificación        |
| **Intereses y crecimiento**       | Declarar intereses, recomendaciones de ruta de crecimiento, roadmaps sugeridos      |
| **Aprendizaje**                   | Asignaciones del manager/instructor, roadmaps, quizzes, progreso                    |
| **OKRs**                          | Objetivos (carrera, habilidad, proyecto, aprendizaje), resultados clave, check-ins  |
| **Desempeño**                     | Autoevaluaciones, evaluaciones ascendentes (evaluar a tu manager)                   |
| **Proyectos**                     | Ver asignaciones, roles, fechas, clientes, miembros del proyecto                    |
| **Buscador de personas**          | Búsqueda por habilidad/capacidad con IA, puntajes de coincidencia, lenguaje natural |
| **Base de conocimiento**          | Perfiles de rol, roadmaps, capacitaciones, documentación de capacidades             |
| **Directorio y organigrama**      | Explorar el equipo, vista de jerarquía, búsqueda por nombre/habilidad               |
| **Feedback y reconocimiento**     | Dar feedback, enviar reconocimiento/elogios con categorías                          |
| **Asistente de IA**               | Chat, buscar personas por habilidades, recomendaciones                              |

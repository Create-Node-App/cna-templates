# Guía del miembro

Como **miembro**, podés gestionar tu perfil, ver tu dashboard e interactuar con el asistente de IA. Esta guía cubre lo disponible para cada miembro de la plataforma.

## Qué podés hacer

| Área                       | Qué tenés disponible                                 |
| -------------------------- | ---------------------------------------------------- |
| **Dashboard**              | Estadísticas, acciones rápidas, resumen de actividad |
| **Perfil y configuración** | Editar perfil, avatar, zona horaria, preferencias    |
| **Asistente de IA**        | Chat para ayuda, búsqueda y recomendaciones          |

## Enlaces rápidos

- [Dashboard](/docs/es/member/dashboard) — Tu base: estadísticas y acciones rápidas
- [Perfil y configuración](/docs/es/member/profile-settings) — Mantener tu perfil al día
- [Asistente de IA](/docs/es/member/ai-assistant) — Obtener ayuda y buscar en lenguaje natural

> **Tip:** Guardá tu Dashboard y usá el Asistente de IA para respuestas rápidas y orientación.
